# 小红书AI内容研究报告

## 1. 什么AI话题在小红书上最火？

小红书上关于AI的笔记已超过**1615万条**，甚至超过了美妆类笔记数量。2025年上半年，科技类内容在小红书上增长近**150%**，阅读量增长超过**200%**。

最受欢迎的AI话题方向：

- **AI工具推荐和教程**：如DeepSeek使用指南、AI绘画工具评测、AI写作工具对比。这类"干货型"内容收藏率极高，因为小红书算法把"收藏"作为内容价值最高指标。
- **AI绘画/图片生成**：Midjourney、Stable Diffusion等工具的作品分享和教程，视觉冲击力强，容易引发传播。
- **AI赚钱/副业**：用AI做小红书运营、AI接单变现等话题，迎合"普通人也能靠AI赚钱"的心理需求。
- **AI创业冷启动**：Manus、Flowith、StressWatch、PlanCoach等AI产品都选择在小红书完成从0到1的冷启动，利用平台的真实用户反馈机制。
- **DeepSeek相关内容**：DeepSeek爆火后，小红书上涌现大量使用分享和教程，某素人账号通过DeepSeek拆解短剧脚本，7天产出38条视频，单周带货佣金突破2.8万元。

## 2. 成功的AI帖子面向什么受众？

**主要面向普通用户，而非技术人群。** 小红书的核心用户画像是年轻女性为主，因此成功的AI内容通常需要降低技术门槛。

小红书官方推出的AI话题策略也印证了这点：
- **#AI新手村**：涵盖AI工具推荐、应用教程、原理知识科普、使用体验
- **#我的AI好奇怪**：偏娱乐向，展示AI的有趣输出

成功的AI内容创作者（如AI绘画博主灵琪）并非以技术深度取胜，而是以"好看的作品 + 简单的教程"打动用户。知识科普类账号通过AI生成问题解答，一年积累6万粉丝；性格测试类账号通过AI生成性格解析，半年涨粉近4万。

**关键洞察**：技术人群会来小红书看AI内容，但爆款内容的受众一定是大众化的。需要把技术话题"翻译"成普通人能理解、能行动的语言。

## 3. 流行AI帖子的语气和风格

小红书的SNS数据有三大特征：
- **高度非规范化**：缩写、谐音梗、跨语种表达层出不穷
- **强上下文依赖**：同一句话在不同话题或社群文化中意义迥异
- **显著情绪化**：用户情感张力随热点事件剧烈波动

成功的AI帖子风格特点：

### 口语化、生活化
- 用第一人称叙述，像朋友聊天
- 正文控制在700字左右
- 标题控制在20字以内
- 大量使用emoji（但不要过度）

### 避免纯技术写法
- 小红书用户对"AI味"（明显AI生成的内容）越来越敏感
- 成功的创作者会学习对标博主的口语节奏、段落长度和emoji用法
- 过于主观性或情绪化的语料反而会在互动中起负面效果

### 爆款内容公式
算法驱动的预推广机制：每篇新笔记会先推送给100-500个用户，前1-4小时内的互动数据决定是否进入更大的流量池。因此：
- 标题要有"钩子"（数字、情绪词、好奇心缺口）
- 开头要快速抓住注意力
- 提供可操作的价值（教程、工具推荐、经验分享）

## 4. "AI创业"、"vibe coding"、"AI工具"话题的表现

### AI创业
表现很好。小红书为AI创业团队提供三大优势：创作者聚集、真实用户需求、即时反馈机制。越来越多AI产品选择在小红书做冷启动。典型案例：PlanCoach的爆款笔记引发了数十万拖延症用户的共鸣，迅速转化为下载和使用。

### Vibe Coding
这是一个新兴但有潜力的话题。已有案例：
- "不会编程的设计师，用Claude Code 2天做了个小红书评论回复AI插件"——这类"非技术人员用AI编程"的故事容易引发共鸣
- Vibe Coding工具（Cursor、Replit、Windsurf等）的对比评测也有流量

### AI工具
表现最好的品类之一。具体方向包括：
- AI写作工具（七燕AI、图小玉等训练数据涵盖1200万+历史爆款文案）
- AI绘画工具（Liblib AI坐拥2500万+用户）
- AI视频工具（萌宠跳舞视频轻松获得几十万点赞）
- 通用大模型（DeepSeek使用教程是流量密码）
