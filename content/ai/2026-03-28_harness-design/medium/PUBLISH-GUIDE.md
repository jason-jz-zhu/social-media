# Publish Guide: Harness Design Article

## Pre-Publish Checklist

### Images
1. Open each HTML file in Chrome (1200x630 viewport)
2. Screenshot at exact dimensions (use DevTools device mode, 1200x630)
3. Save as PNG:
   - `hero.html` -> `hero.png`
   - `four-runs.html` -> `four-runs.png`
   - `overlay-bug.html` -> `overlay-bug.png`
   - `quote.html` -> `quote.png`

### Medium Setup
- **Title:** I Ran the Same AI Task Four Ways. Here's What Broke.
- **Subtitle:** An experiment in harness design: same model, same tools, same requirement -- four architectures, four very different outcomes.
- **Publication:** Self-publish (or submit to Towards Data Science / Better Programming if accepted)
- **Tags:** Artificial Intelligence, AI Agents, Software Engineering, Productivity, Machine Learning
- **SEO Title:** I Ran the Same AI Task Four Ways. Here's What Broke.
- **SEO Description:** Same AI model, same tools, same 2-sentence requirement. Four different architectures produced wildly different results. The notification overlay bug tells the whole story.

### Image Placement
1. **Hero** -- top of article (set as featured image)
2. **Four-runs comparison** -- after "Results: The Four Runs" heading, before the data table
3. **Overlay bug** -- after "The Notification Overlay Bug" heading
4. **Quote card** -- can be used for social sharing or placed before the conclusion

### LinkedIn
- Post `post.md` content as plain text
- After posting, add `comment.md` as first comment with the Medium link
- Best posting time: 18:30 ET (weekday)
