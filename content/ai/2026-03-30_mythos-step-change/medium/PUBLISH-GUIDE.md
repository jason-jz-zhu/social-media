# Publish Guide: Claude Mythos Step Change Article

## Pre-Publish Checklist

### Images
1. Open each HTML file in Chrome (1200x630 viewport)
2. Screenshot at exact dimensions (use DevTools device mode, 1200x630)
3. Save as PNG:
   - `hero.html` -> `hero.png`
   - `model-evolution.html` -> `model-evolution.png`
   - `harness-recalibration.html` -> `harness-recalibration.png`
   - `quote.html` -> `quote.png`

### Medium Setup
- **Title:** Claude Mythos Leaked. Here's What a $450/Month Power User Is Actually Thinking About.
- **Subtitle:** It's not "how good is it." It's "what can I attempt now that I couldn't before."
- **Publication:** Self-publish (or submit to Towards Data Science / Better Programming if accepted)
- **Tags:** Artificial Intelligence, AI Agents, Claude AI, Productivity, Machine Learning
- **SEO Title:** Claude Mythos Leaked. Here's What Step Changes Actually Mean for Practitioners.
- **SEO Description:** Anthropic leaked Claude Mythos. A $450/month Claude power user analyzes what "step change" means for harness design, AI workflows, and what to rebuild when the model improves.

### Image Placement
1. **Hero** -- top of article (set as featured image)
2. **Model evolution** -- after "What 'Step Change' Has Actually Meant" heading
3. **Harness recalibration** -- after "Harness Design Doesn't Become Obsolete" heading
4. **Quote card** -- can be used for social sharing or placed before the conclusion
