配置步骤

两分钟配好，就三步

1
安装 Claude Code
终端一行命令搞定
$ npm install -g @anthropic-ai/claude-code
+ claude-code installed
$ claude
Claude Code v2.1 ready
2
开启 Remote Control
在Claude Code里输入命令，生成连接链接
> /remote-control
Remote Control is active.
Code at claude.ai/code/session_...
Press space for QR code
3
手机连接
扫二维码 或 打开 claude.ai/code 同账号登录
Connected to local session
Model: Claude Opus 4.6
Project: ~/my-project
> ready for input_

需要：Claude Max订阅 ($200/月) 或 Pro ($20/月)。电脑保持开机即可，不用一直盯着。
