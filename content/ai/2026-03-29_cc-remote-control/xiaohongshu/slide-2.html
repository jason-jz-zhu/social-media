Remote Control是什么？

痛点

Opus跑一个任务要好几分钟，你就坐电脑前干等着。
想去倒杯咖啡？不行，等会要approve。

解法

Remote Control = 手机遥控电脑上的Claude Code
审批、查看进度、追加指令，全在手机上完成。
电脑继续跑，你该干嘛干嘛。

工作原理
手机
发指令/审批
→
Anthropic
中转/不存代码
→
电脑
执行代码
关键特性
代码不上传 — 你的代码始终留在本地电脑
一次性凭证 — 每次连接生成新token
不开端口 — 电脑不暴露任何网络端口

做了10年tech的体感：这种"人不在电脑前但活在干"的体验，之前只有CI/CD给过我。
