# Outline: 6点起床学创业 Day 3

## Source
YC "How to Start a Startup" Lecture 3 — Paul Graham
Topics: Counterintuitive Parts of Startups, How to Have Ideas

## Core thesis
创业就像滑雪——直觉全是错的。最好的创业点子不是"想"出来的，是在好奇心驱动下自然冒出来的。

## Key arguments
1. 创业中直觉会害你，除了看人
2. 你不需要创业专家，你需要用户专家（Growth Hacks = bullshit）
3. Gaming the system在创业中不管用（用户像鲨鱼，骗不了）
4. 不要刻意想创业点子——最好的都是side project
5. PG的终极建议：Just Learn

## Personal connections
- 好的点子都是平时突然想到的，但其实之前就注意到了——验证了PG的"side project"论
- 公司里也可以"创业"，但比自己创业简单，因为公司给了资源（intrapreneurship）
- 不要在大学创业——PG是怕孩子们退学吧 哈哈

## Golden quote
用户就像鲨鱼——太蠢了骗不了，要么是肉要么不是
