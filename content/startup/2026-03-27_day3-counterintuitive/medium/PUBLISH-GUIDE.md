# Publish Guide — Day 3: Counterintuitive Parts of Startups

## How to Publish

1. Open Medium and create a new story
2. Copy the content from `article.md` into a markdown preview tool (e.g., Typora, StackEdit, or VS Code preview)
3. Paste the rendered content into Medium's editor
4. Adjust formatting: ensure headings, bold text, and horizontal rules render correctly
5. Add images at the designated locations (see below)
6. Set the preview image to `hero.png`
7. Add topics before publishing

## Image Upload Locations

Upload 4 images from the `images/` folder:

| Image | Placement | File |
