# Medium Publishing Guide — Day 1

## How to publish

**Option A (Recommended): Import Markdown**
1. Go to Medium → New Story
2. Click `...` menu → "Import a story"
3. Upload `article.md`
4. Medium will auto-format headings, bold, italic, lists
5. Manually upload images at the marked locations (see below)

**Option B: Copy-paste**
1. Open article.md in any markdown previewer (VS Code preview, or paste into hackmd.io)
2. Copy the rendered output
3. Paste into Medium editor
4. Manually upload images

## Image upload locations

Insert these images in order (drag & drop into Medium editor):

1. **After the subtitle** → `images/hero.png` (will become the article header image)
2. **After "The examples are now legendary:"** → `images/bad-ideas.png`
3. **After the "100 people love" bold quote** → `images/love-vs-like.png`
4. **At the start of "The Only Good Reason" section** → `images/quote.png`
