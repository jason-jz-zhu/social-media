## 循环协议
1. 真跑 skill → 产出 top3
2. 对照清单打分，记录 FAIL 项
3. FAIL → 修 SKILL.md 对应条款 → 重跑受影响步骤
4. 全过 → 记录版本号和日期在本文件底部
