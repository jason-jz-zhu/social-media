## LinkedIn Post Generation

### Input
- shared/outline.md or user provides topic

### Hook Rule
First 2 lines are life-or-death (LinkedIn shows only 2 lines + "...see more")

### Content Formula
Hook (2 lines) → Personal story (3-5 lines) → 1-2 key insights → Gold quote → CTA question → Hashtags (3-5)

### Length
500-1200 words, English

### Format
Plain text only. No markdown rendering. Use emoji for visual breaks. CAPS for emphasis.

### Links
Links go in comments, NOT in post body (algorithm penalty)

### Tone
Professional but human, thought leader style

### From Mother Content
Only take 1-2 angles, reframe for professional/ROI context

### Publishing
- Tue-Thu, 8-10am ET
- 2 posts/week

### Post-Publish
- Reply to ALL comments within 1 hour
- Commenting on others' posts is MORE important than posting (for growth)
