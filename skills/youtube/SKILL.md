## YouTube Video Script Generation

### Input
- shared/outline.md or user provides topic

### Language
Chinese (中文频道)

### Format
真人出镜 (talking head), 5-15 minutes

### Script Structure
1. Hook (前15秒 — 必须抓人, YouTube算法看完播率)
2. 问题/背景 (30秒)
3. 核心内容 (3-5个要点, 每个1-2分钟)
4. 个人经验/故事
5. 总结 + CTA (订阅/评论)

### Thumbnail
1280x720, 16:9. Face + emotion + 3-5个大字. High contrast.

### Title
≤60 chars, 前30字最重要 (mobile truncation)

### Description
前2行最重要 (above fold), include timestamps, links to other platforms

### Tags
10-15 relevant tags

### From Mother Content
Rewrite as spoken Chinese, more casual than XHS text

### Publishing
- Weekly, same day/time for consistency

### Priority
Last platform to add (month 4-5), highest production cost
