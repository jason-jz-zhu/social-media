## YouTube Platform Rules

- Thumbnail: 1280x720, 16:9, <2MB
- Title: ≤60 chars (≤30 on mobile)
- Description: 5000 char max, first 2 lines above fold
- Tags: 10-15
- Chapters: use timestamps in description
- Cards + end screens for cross-promotion
- First 48 hours determine algorithmic push
- Retention rate is king metric
- 前15秒 completion rate determines everything
