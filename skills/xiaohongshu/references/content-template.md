# content.md Template

Use this exact structure for every Xiaohongshu post's `content.md` file.

```markdown
# 小红书帖子

## 标题
{title — ≤20 characters}

## 正文

{body text — ≤1000 characters, conversational Chinese, no markdown bold, no bullet lists}

## 话题标签
{#Tag1 #Tag2 #Tag3 ... — 5-8 tags, space-separated}

## 封面图片
- cover.png (1080x1440, 3:4比例)
- cover.html (源文件，可修改设计)
```

## Writing Guidelines

- First person, casual tone — like talking to a friend
- Open with a relatable hook (personal experience or observation)
- Build to a core insight or realization
- Close with an invitation to engage (e.g., 评论区聊聊)
- Use line breaks generously for readability on mobile
- Emoji sparingly — 1-2 max, typically at the end
- No markdown formatting in body (no `**bold**`, no `- lists`)
- Plain text only — what you write is what gets pasted into Xiaohongshu
