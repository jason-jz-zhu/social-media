# Xiaohongshu Platform Rules & Constraints

## Character Limits

| Element | Limit |
|---------|-------|
| 标题 (Title) | ≤ 20 characters |
| 正文 (Body) | ≤ 1000 characters |

Chinese characters, punctuation, and emoji each count as 1 character.

## Content Formatting

- **Plain text only** — Xiaohongshu does not render markdown. No `**bold**`, no `# headings`, no `- bullet lists`.
- Line breaks are preserved and encouraged for readability.
- Emoji are supported but use sparingly (1–2 per post).

## Hashtags (话题标签)

- Prefix each tag with `#` (e.g., `#AI时代`)
- 5–8 tags per post is the sweet spot
- Mix broad tags (high discovery) with niche tags (targeted audience):
  - Broad: `#个人成长`, `#AI时代`, `#自我探索`
  - Niche: `#普通人的AI生活`, `#真实表达`
- Keep tags relevant — irrelevant tags hurt distribution

## Cover Image (封面图片)

| Property | Value |
|----------|-------|
| Aspect ratio | 3:4 |
| Resolution | 1080 x 1440 px |
| Format | PNG |

- The cover image is the single biggest driver of click-through rate
- Text on cover should be large, readable, and convey the post's core message
- Warm, aesthetic visuals perform well on the platform
- Avoid cluttered designs — clean and modern wins

## Content Style

- Conversational, authentic tone — like sharing with friends
- First-person narrative performs best
- Vulnerability and real experiences resonate more than polished advice
- End with a question or call-to-action to drive comments
