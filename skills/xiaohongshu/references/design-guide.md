# Cover Image Design Guide

Design system for Xiaohongshu cover images. All covers are self-contained HTML files (1080 x 1440 px) that get screenshotted to PNG.

## Design Principles

1. **3 秒法则** — 用户在信息流中 0.3-3 秒内决定是否点击，视觉冲击力优先
2. **缩略图优先** — 信息流中封面约缩小 3.5 倍显示（~300x400px），所有元素必须在缩略图下可读
3. **高对比度** — 文字与背景明度差 > 30%，强反差封面比单色封面点击率高 40%+
4. **每篇封面视觉必须不同** — 不同的背景色/布局/视觉元素，避免信息流中看起来一样
5. **暖色优先** — 数据显示暖色调（红/橙/黄）整体点击率优于冷色
6. **有视觉焦点** — 纯文字不够，必须有插图/涂鸦/图标等视觉元素

## 封面 vs 内容页的区别（重要）

| | 封面（slide 1） | 内容页（slide 2-8） |
|---|---|---|
| 作用 | 在信息流中竞争点击 | 用户已进入，阅读内容 |
| 背景 | **深色/高饱和色**（必须醒目） | 奶白 #FFF9F5（舒适阅读） |
| 视觉元素 | **必须有**插图/涂鸦/图标 | 可选，保持简洁 |
| 标签药丸 | **不要放**（缩略图看不到，浪费空间） | 仅最后一页放 |
| 标题占比 | **60%+** 画面面积 | 正常排版 |

## Cover Background Strategy

**不再使用奶白底做封面。** 白底在信息流中太安静，容易被跳过。

### 背景模板库（每篇封面从中选不同的）

**模板 A：深色渐变**
```css
background: linear-gradient(160deg, #1A1A2E 0%, #16213E 40%, #0F3460 100%);
/* 主标题：白色 #FFFFFF，强调词：荧光黄 #FFE135 或暖橙 #FF6B35 */
```

**模板 B：暖色渐变**
```css
background: linear-gradient(160deg, #FF6B35 0%, #FF8E53 50%, #FFC371 100%);
/* 主标题：白色 #FFFFFF，强调词：深色 #1A1A1A */
```

**模板 C：高饱和色块**
```css
background: #FF6B6B; /* 或 #FFE135, #00B4D8, #7C3AED 等 */
/* 主标题：白色或深色，与背景形成最强对比 */
```

**模板 D：深色 + 局部高亮色块**
```css
background: #1A1A1A;
/* 标题中的关键词放在一个高饱和色块里（如橙色圆角矩形），制造焦点 */
```

### 选择规则
- 同一系列内的帖子，使用不同模板，避免视觉重复
- 优先选择与话题情绪匹配的背景（焦虑→深色，兴奋→暖色，干货→色块）
- 暖色背景表现最好（橙色帖子 43 浏览 vs 青色帖子 23 浏览）

## Typography

Import from Google Fonts:
```css
@import url('https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@300;400;500;700;900&family=Noto+Serif+SC:wght@400;600;700;900&display=swap');
```

### Headline (主标题)
- Font: `Noto Sans SC`, weight **900**（无衬线更有力量感）
- Size: **100-120px**（尽可能大，占画面 60%+）
- Line height: 1.2
- Color: 与背景形成最强对比（深底白字，亮底深字）
- Letter spacing: 2px

### 关键词高亮（荧光标记笔效果）
```css
.keyword {
  position: relative;
  display: inline;
}
.keyword::after {
  content: '';
  position: absolute;
  bottom: 0;
  left: -6px;
  width: calc(100% + 12px);
  height: 40%;
  background: rgba(255, 107, 53, 0.6); /* 或其他高饱和色 */
  border-radius: 4px;
  z-index: -1;
}
```
- 关键词用荧光标记笔效果而非简单换色
- 标记笔颜色要与背景形成对比

### Subtitle (副标题)
- Font: `Noto Sans SC`, weight **500**
- Size: **48-56px**
- Color: 略低于主标题亮度（深底用 `#CCCCCC`，亮底用 `#555`）
- 放在标题下方，间距 40px

## Visual Elements（封面必须有）

每张封面必须包含至少一种视觉元素：

### 手绘涂鸦风格（推荐）
- 与话题相关的简笔画（灯泡、箭头、星星、对话气泡等）
- 白色或亮色描边，线宽 4-6px
- 大小 150-250px
- 放在标题周围作为装饰，不要挡住文字
- 手绘感 > 精致感（亲近、真实）

### Emoji 放大
- 将一个与话题相关的 emoji 放大到 200px+ 作为视觉焦点
- 放在标题上方或旁边

### AI 生成插图
- 简单的扁平化插图风格
- 与话题直接相关

### 截图/对比图
- 如果内容涉及工具或产品，可以放一个模糊的截图做背景

## Layout Structure（封面专用）

```
┌─────────────────────────┐
│                         │
│    [视觉元素/涂鸦]      │
│    (150-250px)          │
│                         │
│  ████████████████████   │
│  ██ 主标题（超大）██   │
│  ██ 关键词带荧光笔 ██  │
│  ████████████████████   │
│                         │
│     副标题（简短）      │
│                         │
│                         │
└─────────────────────────┘
```

- **没有标签药丸**（省空间，标签放正文里）
- **没有底部装饰 art**（省空间）
- **没有分割线**（封面不需要）
- 标题占画面 60%+，越大越好
- 内容垂直居中偏上（小红书信息流底部会被浏览数遮挡）

## Background Glow Shapes

- 1-2 个大的 radial-gradient 色块，blur(100px)，opacity 0.3-0.5
- 颜色与主色调呼应但不抢焦点
- 制造深度感和氛围

## Grain Texture
- SVG noise overlay at `opacity: 0.04`
- 给画面增加质感，避免纯色太"数码"

## Adapting for New Topics

1. **从背景模板库中选一个不同于最近发布的模板** — 避免信息流视觉重复
2. 标题和副标题匹配话题
3. 选择与话题相关的视觉元素（涂鸦/emoji/插图）
4. 关键词加荧光标记笔效果
5. **缩略图自检** — 完成后缩小到 300x400 检查：标题能读吗？视觉焦点明显吗？

## Common Mistakes to Avoid

- ❌ 用奶白/浅色做封面背景（信息流中太安静）
- ❌ 所有封面用同一个模板（无法区分）
- ❌ 纯文字无视觉元素（缺乏钩子）
- ❌ 放标签药丸（缩略图看不到，浪费空间）
- ❌ 标题太小（<90px 缩略图不可读）
- ❌ 封面和内容页用同一种背景（封面需要更强冲击力）
- ❌ 使用冷色调（青色/蓝绿色表现最差）
