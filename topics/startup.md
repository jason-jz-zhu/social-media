# 创业系列话题池（6点起床学创业）

## 系列定位
基于 YC "How to Start a Startup" (Stanford CS183B) 20节课的学习笔记。每天早上6点起床，在孩子醒之前学一节课，当天分享笔记和思考。

## 系列信息
- 课程：How to Start a Startup (CS183B, Stanford / Y Combinator, 2014)
- 组织者：Sam Altman
- 总计：20节课，每节约50分钟
- 强调色：紫色 #7C3AED
- 小红书系列名：6点起床学创业 Day N
- LinkedIn系列名：What YC Teaches About Startups (Day N/20)
- Medium系列名：20 Days of YC's Startup Bible
